# My Todo List (ウィジェット連携型 Todoアプリ)

Android向けのシンプルなTodoリストアプリです。
Jetpack ComposeでUIを構築し、Glanceを使用してホーム画面用のウィジェットを実装しています。

## 📱 主な機能
- **タスクの追加・削除**: 優先度（高・中・低）と目標時間を設定してタスクを管理できます。
- **自動ソート機能**: タスクは「未完了が上（完了は下）」「優先度順（高→中→低）」「時間順」で自動的に綺麗に並び替わります。
- **ホーム画面ウィジェット連携**: アプリを開かなくても、ホーム画面から今日のタスクを一覧で確認したり、完了チェックを入れたりできます。
- **データ永続化**: アプリを閉じたり、端末を再起動したりしても、タスクデータが消えずに保持されます。

## 🛠️ 開発環境 / 技術スタック
- **OS**: Android 13以上推奨
- **言語**: Kotlin
- **UI Framework**: Jetpack Compose
- **Widget**: AndroidX Glance (SharedPreferencesによる高速同期)
- **ビルドシステム**: Gradle (Kotlin DSL)
